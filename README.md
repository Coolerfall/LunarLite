LunarLite
===========================
A lite Chinese lunar calendar App. This app is based on [Android-LunarView][1].


Libraries Used
==============
* [Dagger2][2]
* [RxJava][3]
* [RxAndroid][4]
* [Butter Knife][5]
* [Phrase][6]
* [SqlBrite][7]
